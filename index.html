  1. Download the browser extension

    Download the Guidance browser extension zip file. File size is approximately 20KB.

    Download Browser Extension

  2. Unzip the file

    Right-click the downloaded zip → Extract All → Extract. Move the extracted folder somewhere permanent like C:\Users\YourName\guidance-browser. Do not delete this folder after installing.

  3. Load into Edge or Chrome

    Open edge://extensions or chrome://extensions → Toggle Developer mode ON → Click Load unpacked → Select the extracted folder → Click Select Folder.

  4. Pin Guidance to your toolbar

    Click the puzzle piece 🧩 icon in your browser toolbar → Find Guidance — Visual Coding Co-pilot → Click the pin icon.

  1. Install VS Code (required)

    Download VS Code free from code.visualstudio.com. Requires approximately 350MB of free space.

  2. Install Node.js (required)

    Download the LTS version from nodejs.org. Choose the Windows installer. Skip if already installed.

  3. Download the VS Code extension

    Download the Guidance VS Code extension zip file below. File size is approximately 10KB.

    Download VS Code Extension

  4. Install the VS Code extension

    Unzip the downloaded file. Open VS Code → press Ctrl+` to open the Terminal → run these two commands one at a time:

    xcopy /e /i /y "C:\Users\YourName\Downloads\VSCODE-guidance-extension-NEW" "C:\Users\YourName\.vscode\extensions\guidance-vscode"

    Then: cd "%USERPROFILE%\.vscode\extensions\guidance-vscode" && npm install ws

    Close and reopen VS Code. You'll see Guidance: ready in the bottom status bar.

  5. Open your project in VS Code

    In VS Code: File → Open Folder → select your project folder. The status bar will show Guidance: browser connected once the browser extension is also running.

  6. Assign project colours (optional)

    Press Ctrl+Shift+P → type Guidance: Assign Colour → select your project → pick a colour.

Common questions

Does Guidance work on Mac or iOS?

Not yet. Windows 10 and 11 only. Mac and iOS support is planned for a future release.

Do I need VS Code? Can I use Notepad++ instead?

Yes — VS Code is required. It needs to communicate directly with the editor to show highlights. VS Code is free and opens all the same file types as Notepad++.

How much storage space do I need?

Approximately 800MB total: VS Code (~350MB), Node.js (~300MB), and the extensions (~5MB). If you already have VS Code and Node.js, you only need 5MB.

Do I need to understand coding to use Guidance?

No. Guidance is built for people who use AI to write code but struggle with where to apply the changes. You don't need to understand the code — Guidance shows you exactly where to paste it.

Which AI platforms does it work with?

ChatGPT (chatgpt.com), Claude (claude.ai), Grok (grok.com) and Gemini (gemini.google.com). It works with all of them simultaneously.

Is my code sent anywhere?

No. Everything runs on your local machine over a local WebSocket connection — your code never leaves your computer.
