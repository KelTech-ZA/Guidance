Installation

Up and running in 5 minutes.

Two things to install — the browser extension and the VS Code extension.

💻 System Requirements
Operating System
Windows 10 or 11
Mac / iOS — coming soon
Browser
Microsoft Edge or Chrome
Firefox not supported yet
Storage Required
~800MB free space
VS Code ~350MB · Node.js ~300MB · Extensions ~5MB
1
Download the browser extension
Download the Guidance browser extension zip file. File size is approximately 20KB.
Download Browser Extension
2
Unzip the file
Right-click the downloaded zip → Extract All → Extract. Move the extracted folder somewhere permanent like C:\Users\YourName\guidance-browser. Do not delete this folder after installing.
3
Load into Edge or Chrome
Open edge://extensions or chrome://extensions → Toggle Developer mode ON → Click Load unpacked → Select the extracted folder → Click Select Folder.
4
Pin Guidance to your toolbar
Click the puzzle piece 🧩 icon in your browser toolbar → Find Guidance — Visual Coding Co-pilot → Click the pin icon. The Guidance icon will now appear in your toolbar permanently.
5
Activate Pro (if purchased)
Click the Guidance icon in your toolbar → Go to the Settings tab → Paste your licence key in the Activate Pro box → Click Activate. Pro features unlock immediately.
1
Install VS Code (required)
Guidance requires VS Code as your code editor — it cannot work with Notepad++ or other editors. Download VS Code free from code.visualstudio.com. It requires approximately 350MB of free space.
2
Install Node.js (required)
Download the LTS version from nodejs.org. Choose the Windows installer. This requires approximately 300MB of free space. If you already have Node.js installed, skip this step.
3
Download the VS Code extension
Download the Guidance VS Code extension zip file below. File size is approximately 10KB.
Download VS Code Extension
4
Install the VS Code extension
Unzip the downloaded file. Open VS Code → press Ctrl+` to open the Terminal → run these two commands one at a time:

xcopy /e /i /y "C:\Users\YourName\Downloads\VSCODE-guidance-extension-NEW" "C:\Users\YourName\.vscode\extensions\guidance-vscode"

Then: cd "%USERPROFILE%\.vscode\extensions\guidance-vscode" && npm install ws

Close and reopen VS Code. You'll see Guidance: ready in the bottom status bar.
5
Open your project in VS Code
In VS Code: File → Open Folder → select your project folder. The status bar at the bottom will show Guidance: browser connected once you also have the browser extension running on a ChatGPT, Claude, Grok or Gemini tab.
6
Assign project colours (optional)
To colour-code your project tabs: press Ctrl+Shift+P in VS Code → type Guidance: Assign Colour → select your project → pick a colour. Your VS Code title bar and tabs will update immediately.
