Features

Everything you need.
Nothing you don't.

🔍

AI-Powered Patch Detection

Guidance monitors ChatGPT, Claude, Grok, and Gemini simultaneously. The moment an AI outputs a code block, it's captured — including file names, language detection, and context. No more copy-paste. No more lost patches.

✓ All 4 AI platforms
🎨

Tab Colour Coding

Assign custom colours to domains. ChatGPT tabs glow green, Claude tabs glow orange. Find the right tab at a glance without reading a word.

✓ Fully customisable
📋

Patch Queue

All detected code patches in one scrollable list. Apply them one by one or all at once. Patches persist across sessions so nothing is ever lost.

✓ Persistent storage
🔌

Local WebSocket Server

Connects to a local server on port 8765 so patches can be sent directly to your IDE or code editor. Zero latency, zero cloud dependency.

✓ Works offline
🛡️

Privacy First

Your code never leaves your machine. All processing happens locally in your browser. No external servers. No analytics on your patches.

✓ 100% local
⚡

Instant Language Detection

Python, JavaScript, TypeScript, HTML, CSS, Java, C# — Guidance identifies the language of every patch automatically so you always know what you're applying.

✓ 10+ languages
Install Guide

Up and running in 60 seconds

No account. No payment. No nonsense.

1

Download the extension

Click the button below to download the Guidance .zip file.

2

Extract the zip

Unzip the downloaded file to a folder you won't accidentally delete.

3

Open Chrome Extensions

Navigate to chrome://extensions in your browser.

4

Enable Developer Mode

Toggle "Developer mode" in the top right of the extensions page.

5

Load Unpacked

Click "Load unpacked" and select the unzipped Guidance folder.

6

Start coding!

Pin Guidance to your toolbar and open any AI chat to see it in action.
